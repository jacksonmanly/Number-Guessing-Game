# Number-Guessing-Game
A C++ console game I built while learning C++.

## Features

- Easy, Medium and Hard difficulty
- Input validation
- Attempt counter
- Play again option
- - Random number generation

## Concepts Used

- Variables
- Functions
- if / else if
- while loops
- User input
- Nested loops
